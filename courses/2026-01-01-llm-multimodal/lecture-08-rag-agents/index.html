üß© What Is Retrieval-Augmented Generation (RAG)?

RAG = Knowledge + Reasoning

Instead of forcing the model to:

  • memorize everything
  • hallucinate confidently

We let it:

  1. Retrieve relevant information
  2. Reason over it
  3. Generate grounded answers

üß† Embeddings: The Heart of RAG

Embedding models map meaning ‚Üí vectors.

Examples:

  • Text: sentence-transformers
  • Image: CLIP
  • Video: InternVideo
  • Document: Layout-aware embeddings

Good retrieval beats bigger models.

üêç Python: Minimal RAG Example

query = "What is transformer attention?"

q_emb = embedder.encode(query)
docs = vector_db.search(q_emb, top_k=5)

context = "\n".join(docs)

answer = llm.generate(
    prompt=f"Answer using the context below:\n{context}\n\nQuestion:{query}"
)

‚ö†Ô∏è Common RAG Failure Modes

  • Retrieving irrelevant chunks
  • Context too long
  • Context ignored
  • Conflicting documents
  • Over-trusting retrieved text

Mitigation:

  • Chunking strategy
  • Reranking
  • Instruction tuning
  • Answer verification

ü§ñ What Is an AI Agent?

An agent is an LLM that can act.

Agent abilities:

  • Decide next action
  • Use tools
  • Store memory
  • Observe outcomes
  • Iterate

üß† Agent Loop (Canonical)

Observe ‚Üí Think ‚Üí Act ‚Üí Reflect ‚Üí Repeat

This is not prompting ‚Äî it is control flow.

üß† What Is Agentic AI?

Agentic AI means:

  • Long-horizon goals
  • Autonomous planning
  • Self-correction
  • Tool orchestration
  • Memory persistence

Examples:

  • Research agents
  • Coding agents
  • Multimodal assistants
  • Auto-analysts
