Lecture 02 ‚Äî Python: The Language of AI

~2‚Äì3 hours (core recap lecture)

üêç Why Python?

Python is the language of thinking for AI and algorithms.

It is used in:

  • ü§ñ Artificial Intelligence
  • üìä Data Science
  • üß† Research & prototyping
  • üß© Coding interviews (LeetCode, Google, Meta)

If you know Python well, you can think clearly.

üß† Mental Model (IMPORTANT)

Before syntax, remember this:

Python executes line by line, top to bottom.
Variables point to objects.
Everything is an object.

This mental model helps you avoid 80% of bugs.

1Ô∏è‚É£ Variables & Data Types

üì¶ Variables

x = 10
name = "AI"
pi = 3.14
is_active = True

Python is dynamically typed:

x = "now a string"

üî¢ Core Data Types

Type Example
int 10
float 3.14
str "hello"
bool True, False
None None

‚ùì Quiz

Is Python statically typed?

No. Python is dynamically typed.

2Ô∏è‚É£ Control Flow (if / else)

x = 5

if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")

üß† Interview Tip

Indentation is logic in Python.

‚ùì Quiz

What happens if indentation is wrong?

The program raises an IndentationError or behaves incorrectly.

3Ô∏è‚É£ Loops (for / while)

üîÅ for-loop (most common)

for i in range(5):
    print(i)

üîÑ while-loop

count = 0
while count < 3:
    print(count)
    count += 1

üß† Interview Tip

Use for unless you must use while.

‚ùì Quiz

When is while-loop dangerous?

When the stopping condition is incorrect ‚Üí infinite loop.

4Ô∏è‚É£ Lists (Most Important for LeetCode)

nums = [1, 2, 3, 4]

Common operations

nums.append(5)
nums.pop()
nums[0]
nums[-1]
len(nums)

Looping over list

for n in nums:
    print(n)

With index

for i, n in enumerate(nums):
    print(i, n)

‚ùì Quiz

What is the time complexity of list append?

Amortized O(1)

5Ô∏è‚É£ Strings (Text = Sequence)

s = "machine"

Strings behave like lists:

s[0]        # 'm'
s[-1]       # 'e'
s[1:4]      # 'ach'

Useful methods

s.lower()
s.upper()
s.split()

‚ùì Quiz

Are strings mutable?

No. Strings are immutable.

6Ô∏è‚É£ Dictionaries (Key‚ÄìValue Thinking)

student = {
    "name": "Kao",
    "age": 25,
    "major": "AI"
}

Access

student["name"]
student.get("age")

Loop

for key, value in student.items():
    print(key, value)

üß† Interview Tip

Dictionaries give O(1) average lookup.

‚ùì Quiz

Why are dictionaries powerful?

Fast lookup using hashing.

7Ô∏è‚É£ Sets (Uniqueness)

nums = [1, 2, 2, 3]
unique = set(nums)

Use cases

  • Remove duplicates
  • Fast membership check
if 3 in unique:
    print("Found")

‚ùì Quiz

Does a set preserve order?

No.

8Ô∏è‚É£ Functions (Reusable Thinking)

Default arguments

‚ùì Quiz

Why use functions?

Abstraction, reuse, clarity.

üîü Time Complexity Awareness (VERY IMPORTANT)

Python coding interviews are algorithm interviews, not syntax tests.
