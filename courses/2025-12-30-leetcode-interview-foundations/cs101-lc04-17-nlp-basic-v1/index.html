üü¢ NLP 1 ‚Äî Word Count (Warm-Up)

Messi speaks:

text = "messi plays football and messi inspires the world"

Task: Count how many times each word appears.

‚úÖ Solution 
counts = {}
for w in text.split():
    counts[w] = counts.get(w, 0) + 1
print(counts)

üü¢ NLP 2 ‚Äî Unique Vocabulary Size

Task: Count how many unique words are in the text.

‚úÖ Solution 
vocab = set(text.split())
print(len(vocab))

üü¢ NLP 3 ‚Äî Capitalization Detection

sentence = "Messi Is The Greatest"

Task: Return all words that start with a capital letter.

‚úÖ Solution 
caps = [w for w in sentence.split() if w[0].isupper()]
print(caps)

üü¢ NLP 4 ‚Äî Lowercase Normalization

Task: Convert text to lowercase and remove extra spaces.

text = "  Messi   Plays   Football "
‚úÖ Solution 
clean = " ".join(text.lower().split())
print(clean)

üü° NLP 5 ‚Äî Character Frequency

Task: Count how often each character appears (ignore spaces).

text = "messi"
‚úÖ Solution 
freq = {}
for c in text:
    if c != " ":
        freq[c] = freq.get(c, 0) + 1
print(freq)

üü° NLP 6 ‚Äî Find Keywords in Text

keywords = ["goal", "win", "champion"]
text = "messi scores a goal to win the match"

Task: Return keywords that appear in text.

‚úÖ Solution 
found = []
for k in keywords:
    if k in text:
        found.append(k)
print(found)

üü° NLP 7 ‚Äî Sentence Length Analyzer

sentences = [
  "Messi plays football",
  "Messi inspires millions of people around the world"
]

Task: Return sentence lengths (in words).

‚úÖ Solution 
lengths = [len(s.split()) for s in sentences]
print(lengths)

üü° NLP 8 ‚Äî Stopword Removal

stopwords = {"the","and","is"}
text = "messi is the best and the greatest"
‚úÖ Solution 
filtered = [w for w in text.split() if w not in stopwords]
print(" ".join(filtered))

üîµ NLP 9 ‚Äî Bigram Generation

text = "messi wins world cup"

Task: Generate word bigrams.

‚úÖ Solution 
words = text.split()
bigrams = [(words[i], words[i+1]) for i in range(len(words)-1)]

üîµ NLP 10 ‚Äî Most Frequent Word

text = "messi messi goal goal goal win"
‚úÖ Solution 
freq = {}
for w in text.split():
    freq[w] = freq.get(w, 0) + 1
print(max(freq, key=freq.get))

üîµ NLP 11 ‚Äî Prefix Matching (Search Autocomplete)

words = ["messi","message","meta","goal"]
prefix = "me"
‚úÖ Solution 
print([w for w in words if w.startswith(prefix)])

üîµ NLP 12 ‚Äî Suffix Detection

Task: Find words ending with "ing".

words = ["playing","played","scoring","score"]
‚úÖ Solution 
print([w for w in words if w.endswith("ing")])

üî¥ NLP 13 ‚Äî Streaming Word Count (Big Data Mindset)

Messages arrive one by one.

stream = ["messi scores", "messi wins", "scores again"]
‚úÖ Solution 
counts = {}
for msg in stream:
    for w in msg.split():
        counts[w] = counts.get(w, 0) + 1
print(counts)

üî¥ NLP 14 ‚Äî MapReduce (Word Count)

Map: emit (word, 1) Reduce: sum values

‚úÖ Solution 
mapped = []
for s in stream:
    for w in s.split():
        mapped.append((w,1))

reduced = {}
for w,v in mapped:
    reduced[w] = reduced.get(w,0) + v

print(reduced)

üî¥ NLP 15 ‚Äî Longest Word in Corpus

text = "messi demonstrates extraordinary football intelligence"
‚úÖ Solution 
words = text.split()
print(max(words, key=len))

üî¥ NLP 16 ‚Äî Named Entity Heuristic

Rule: Words starting with capital = entity.

text = "Messi plays for Argentina"
‚úÖ Solution 
entities = [w for w in text.split() if w[0].isupper()]
print(entities)

üî¥ NLP 17 ‚Äî Sentence Tokenization

text = "Messi scored. Fans celebrated. History written."
‚úÖ Solution 
sentences = [s.strip() for s in text.split(".") if s]
print(sentences)

üî¥ NLP 18 ‚Äî Text Similarity (Bag of Words)

a = "messi scores goals"
b = "messi scores"
‚úÖ Solution 
sa, sb = set(a.split()), set(b.split())
print(len(sa & sb) / len(sa | sb))

üî¥ NLP 19 ‚Äî Detect Shouting (Uppercase Ratio)

text = "GOAL GOAL Messi"
‚úÖ Solution 
caps = sum(1 for c in text if c.isupper())
print(caps / len(text) > 0.5)

üî¥ NLP 20 ‚Äî Real NLP Interview Question

Task: Detect duplicate sentences.

sentences = [
  "messi scores",
  "messi wins",
  "messi scores"
]
‚úÖ Solution 
seen = set()
dups = []

for s in sentences:
    if s in seen:
        dups.append(s)
    seen.add(s)

print(dups)
