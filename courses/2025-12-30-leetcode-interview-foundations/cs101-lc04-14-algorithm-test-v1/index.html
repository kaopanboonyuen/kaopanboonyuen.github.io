üü° Problem 6 ‚Äî Rate Limiter

Requests arrive with timestamps (seconds):

requests = [1, 1, 1, 2, 2, 3, 10]

Rule: Max 3 requests per second.

Task: Return timestamps that violate the rule.

‚úÖ Solution 
from collections import defaultdict

count = defaultdict(int)
violations = []

for r in requests:
    count[r] += 1
    if count[r] > 3:
        violations.append(r)

print(violations)

üîµ Problem 10 ‚Äî Longest Continuous Uptime

1 = up, 0 = down

Task: Find longest continuous uptime.

‚úÖ Solution

for s in status:
    if s == 1:
        cur += 1
        max_up = max(max_up, cur)
    else:
        cur = 0
print(max_up)

üîµ Problem 11 ‚Äî Merge Sorted Event Streams

a = [1,3,5]
b = [2,4,6]

Task: Merge into sorted order without sort().

‚úÖ Solution 
i = j = 0
res = []

while i < len(a) and j < len(b):
    if a[i] < b[j]:
        res.append(a[i]); i += 1
    else:
        res.append(b[j]); j += 1

res.extend(a[i:])
res.extend(b[j:])
print(res)

üî¥ Problem 13 ‚Äî LRU Cache Simulation

Capacity = 2 Access pattern:

Task: Simulate LRU eviction.

‚úÖ Solution 
cache = []
cap = 2

for x in access:
    if x in cache:
        cache.remove(x)
    elif len(cache) == cap:
        cache.pop(0)
    cache.append(x)

print(cache)

üî¥ Problem 15 ‚Äî Shortest Error Window

logs = ["OK","ERR","OK","ERR","ERR","OK"]

Task: Smallest subarray containing 2 ERR.

‚úÖ Solution 
left = 0
err = 0
best = float("inf")

for right in range(len(logs)):
    if logs[right] == "ERR":
        err += 1
    while err >= 2:
        best = min(best, right - left + 1)
        if logs[left] == "ERR":
            err -= 1
        left += 1

print(best)

üî¥ Problem 17 ‚Äî Dependency Resolution (Graph)

deps = {"A":["B"],"B":["C"],"C":[]}

Detect cycles.

‚úÖ Solution 
vis = set()
path = set()

def dfs(n):
    if n in path: return True
    if n in vis: return False
    path.add(n)
    for nei in deps[n]:
        if dfs(nei): return True
    path.remove(n)
    vis.add(n)
    return False

print(any(dfs(x) for x in deps))

üî¥ Problem 19 ‚Äî Median of Data Stream

Stream:

nums = [5,15,1,3]
‚úÖ Solution 
import heapq
low, high = [], []

for n in nums:
    heapq.heappush(low, -n)
    heapq.heappush(high, -heapq.heappop(low))
    if len(high) > len(low):
        heapq.heappush(low, -heapq.heappop(high))

    if len(low) > len(high):
        print(-low[0])
    else:
        print((-low[0] + high[0]) / 2)
