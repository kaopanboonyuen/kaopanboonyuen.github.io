üü° Puzzle 6 ‚Äî Sliding Window CPU Average

‚úÖ Solution

üî¥ Puzzle 14 ‚Äî Deadlock Detection (Graph)

Edges:

edges = {"A":["B"],"B":["C"],"C":["A"]}

Task:
Detect cycle.

‚úÖ Solution 
visited = set()
stack = set()

def dfs(n):
    if n in stack:
        return True
    if n in visited:
        return False
    visited.add(n)
    stack.add(n)
    for nei in edges.get(n,[]):
        if dfs(nei):
            return True
    stack.remove(n)
    return False

print(any(dfs(n) for n in edges))
