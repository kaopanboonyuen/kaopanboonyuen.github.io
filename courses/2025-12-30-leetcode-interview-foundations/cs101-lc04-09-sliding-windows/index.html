LC-TEST-09 ‚Äî Sliding Window Like a Pro

üéØ Why Sliding Window Is a Game Changer

Sliding Window is what happens when:

Two pointers learn to cooperate.

This technique is everywhere:

  • Subarrays
  • Substrings
  • Streaming data
  • Real-time analytics

Interviewers love this topic.

üß† What You Will Learn

This lesson trains you to:

  • Track state using containers (dict, set)

  • Write code that looks effortlessly smart

üß© Core Example ‚Äî Minimum Window Substring

Problem Statement

Given strings s and t, return the minimum window in s that contains all characters of t.

If no such window exists, return "".

üì• Sample Input / üì§ Output

s t Output
“ADOBECODEBANC” “ABC” “BANC”

üß† Interviewer Insight

‚ùó This problem tests:

  • Can you track frequencies
  • Do you know when a window becomes valid
  • Can you shrink without breaking correctness

If you brute force substrings ‚Üí instant fail.

üß© Sliding Window Strategy

üí° Mental Model

  • Right pointer expands the window
  • Left pointer shrinks it
  • A container tracks validity

Never reset. Never restart loops.

üßë‚Äçüíª Sliding Window Solution

from collections import Counter

def minWindow(s, t):
    need = Counter(t)
    missing = len(t)
    left = start = end = 0

    for right, ch in enumerate(s, 1):
        if need[ch] > 0:
            missing -= 1
        need[ch] -= 1

        if missing == 0:
            while left < right and need[s[left]] < 0:
                need[s[left]] += 1
                left += 1

            if end == 0 or right - left < end - start:
                start, end = left, right

            need[s[left]] += 1
            missing += 1
            left += 1

    return s[start:end]

‚è±Ô∏è Complexity

  • Time: O(n)
  • Space: O(k) (character set)

This is optimal and interview-approved.

üß† Sliding Window Patterns

Pattern Example
Fixed window Max sum subarray
Variable window Min window substring
Frequency window Anagrams
Set-based window Longest unique substring

üéØ Final Takeaway

Sliding Window is not hard.

Restarting loops is.

If your window never resets, your solution is probably correct.
