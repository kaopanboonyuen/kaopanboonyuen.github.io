DE101-PD06 ‚Äî Finding Insights with Pandas (From Data to Decisions)

~120 minutes

üéØ Goal of This Chapter

This chapter teaches how to think, not just how to code.

You will learn:

  • How to ask good questions
  • How to extract insights from data
  • How real companies analyze performance
  • How to communicate findings clearly

‚öΩ Mock Dataset ‚Äî FIFA World Cup Player Stats

We will use a simplified dataset inspired by FIFA World Cup matches,
with focus on Lionel Messi.

import pandas as pd

data = {
    "player": ["Messi", "Messi", "Messi", "Mbappe", "Mbappe", "Neymar", "Modric", "Messi"],
    "team": ["Argentina", "Argentina", "Argentina", "France", "France", "Brazil", "Croatia", "Argentina"],
    "match": [1, 2, 3, 1, 2, 1, 1, 4],
    "goals": [1, 2, 1, 2, 1, 1, 0, 1],
    "assists": [0, 1, 0, 0, 1, 0, 1, 0],
    "shots": [4, 6, 5, 7, 5, 4, 2, 3],
    "minutes": [90, 90, 90, 90, 90, 90, 90, 90],
    "position": ["FW", "FW", "FW", "FW", "FW", "FW", "MF", "FW"]
}

df = pd.DataFrame(data)
df

üß† 20 Insight Questions (with Answers)

Q1Ô∏è‚É£ How many matches did each player play?

df.groupby("player")["match"].count()

üìå Insight: Player usage & reliability.

Q2Ô∏è‚É£ Total goals per player?

df.groupby("player")["goals"].sum().sort_values(ascending=False)

üìå Insight: Who is the main scorer?

Q3Ô∏è‚É£ Average goals per match?

df.groupby("player")["goals"].mean()

üìå Insight: Consistency vs volume.

Q4Ô∏è‚É£ Who contributed most (goals + assists)?

df.assign(contribution=df["goals"] + df["assists"]) \
  .groupby("player")["contribution"].sum()

üìå Insight: True attacking impact.

Q5Ô∏è‚É£ Messi‚Äôs total World Cup impact?

df[df["player"] == "Messi"][["goals", "assists"]].sum()

üìå Insight: Player-specific deep dive.

Q6Ô∏è‚É£ Shots-to-goals efficiency?

df.assign(efficiency=df["goals"] / df["shots"]) \
  .groupby("player")["efficiency"].mean()

üìå Insight: Finishing quality.

Q7Ô∏è‚É£ Which team scored the most goals?

df.groupby("team")["goals"].sum()

üìå Insight: Team dominance.

Q8Ô∏è‚É£ Messi‚Äôs share of Argentina goals?

total_arg_goals = df[df["team"] == "Argentina"]["goals"].sum()
messi_goals = df[df["player"] == "Messi"]["goals"].sum()

messi_goals / total_arg_goals

üìå Insight: Team dependency on star player.

Q9Ô∏è‚É£ Goals by position?

df.groupby("position")["goals"].sum()

üìå Insight: Tactical contribution.

Q1Ô∏è‚É£0Ô∏è‚É£ Average shots per match per player?

df.groupby("player")["shots"].mean()

üìå Insight: Offensive involvement.

Q1Ô∏è‚É£1Ô∏è‚É£ Which match had the most goals?

df.groupby("match")["goals"].sum().sort_values(ascending=False)

üìå Insight: High-impact games.

Q1Ô∏è‚É£2Ô∏è‚É£ Messi trend over matches?

df[df["player"] == "Messi"].sort_values("match")[["match", "goals"]]

üìå Insight: Performance trajectory.

Q1Ô∏è‚É£3Ô∏è‚É£ Player with best goals per 90 minutes?

df.assign(goals_per_90=df["goals"] / df["minutes"] * 90) \
  .groupby("player")["goals_per_90"].mean()

üìå Insight: Fair comparison.

Q1Ô∏è‚É£4Ô∏è‚É£ Assist leaders?

df.groupby("player")["assists"].sum().sort_values(ascending=False)

üìå Insight: Playmaking ability.

Q1Ô∏è‚É£5Ô∏è‚É£ Messi vs Mbappe comparison?

df[df["player"].isin(["Messi", "Mbappe"])] \
  .groupby("player")[["goals", "assists", "shots"]].mean()

üìå Insight: Superstar comparison.

Q1Ô∏è‚É£6Ô∏è‚É£ Which team relies most on one player?

team_totals = df.groupby("team")["goals"].sum()
player_totals = df.groupby(["team", "player"])["goals"].sum()

(player_totals / team_totals).sort_values(ascending=False)

üìå Insight: Risk concentration.

Q1Ô∏è‚É£7Ô∏è‚É£ Match-by-match contribution table?

df.assign(contribution=df["goals"] + df["assists"]) \
  .pivot_table(
      index="match",
      columns="player",
      values="contribution",
      aggfunc="sum"
  )

üìå Insight: Game-level impact.

Q1Ô∏è‚É£8Ô∏è‚É£ Who is most consistent scorer?

df.groupby("player")["goals"].std()

üìå Insight: Low variance = consistency.

Q1Ô∏è‚É£9Ô∏è‚É£ Identify underperformers (high shots, low goals)

df.assign(conversion=df["goals"] / df["shots"]) \
  .sort_values("conversion")

üìå Insight: Optimization opportunity.

Q2Ô∏è‚É£0Ô∏è‚É£ Executive Summary Table

df.groupby("player").agg(
    matches=("match", "count"),
    goals=("goals", "sum"),
    assists=("assists", "sum"),
    shots=("shots", "sum")
)

üìå Insight: One-table decision view.
